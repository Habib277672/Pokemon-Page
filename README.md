Pokémon Explorer

A React JS web app that lets users browse and search Pokémon in an interactive, responsive interface.

Features

Search Functionality: Quickly find Pokémon by name.

Dynamic Rendering: Fetches and displays Pokémon data in real-time.

Reusable Components: Built with modular React components for scalability.

Responsive Design: Works seamlessly across devices and screen sizes.

Technologies Used

React JS

CSS

Axios or Fetch API for data retrieval.
